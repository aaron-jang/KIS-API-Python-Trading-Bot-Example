## 3. 매수 로직

### 자율주행 마스터 스��치 (Regime-Switching)

공포지수(SOXX HV, VXN) 가중치를 타격선 곱셈에서 **전면 배제**.
가중치 > 1.0이면 폭락장으��� 판별하여 하방 매수 차단, 상방 익절만 ��동.
타격선은 **1배수 기초지수 1년 ATR의 3배수 절대 ��폭**으로 영구 고정.

### 다중 무결성 캐시 방어망

야후 API 통신 장애 시 직전 영업일에 저장된 1년 ATR 및 HV/VXN 평균값 캐시를 로드(fsync 적용).
콜드 스타트 시에만 보수적 기본값으로 롤백하여 ZeroDivision 방어.

### 자전거래 원천 차단

매수 주문가 >= 매도 최저가일 경��� → 매수가를 `매도 최저가 - $0.01`로 강제 교정.

### 하이브리드 거래량 스나이퍼

1. 당일 고가 대비 고정 타격선(%) 하락 시 안전장치 해제
2. 실시간 5분봉 최저점 갱신 중 양봉 출현 + 바닥 대비 반등 + Vol > MA20 충족 시 방아쇠 장전
3. **Strong Down 필터**: 1분봉 누적 거래량 50% 이상이 VWAP 하단 집중 시 데드캣 바운스 무시
4. **유령 체결 방어**: 주문 후 최대 8초 동안 미체결 원장 + 실제 체결 내역 동시 스캔 (ODNO 기반)

### VWAP 매수

- 종목별 1년 U-Curve 가중치로 장 마감 30분 전 예산 동적 분할
- 실시간 매도 1호가(Ask)로 정밀 타격, 평���가/별값 초과 시 관망 (고점 불타기 차단)
- **Strong Up 보류**: VWAP 상단 거래량 집중 시 매수 전면 중단, LOC 방어선만 유지
- **누적 잔차 이월**: 1주 미만 소수점 수량을 메모리에 누적, 정수 돌파 시 타격
- **리버스 바이패스**: 리버스 모드 시 VWAP 중단, 오리지널 무매 로직으로 회귀

### 에스크로(Escrow) 락다운

- 특정 종목 리버스 진입 시 현금을 가상 장부에 격리 → 타 종목 예산 탈취 차단

### 예산 소진 동적 탈출

- 실질 가용 예산 < 1회분 또는 보유 수량 < 4주 → 리버스 모�� 전환
